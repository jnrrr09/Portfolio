# Personal Portfolio

This repository contains the source code for my personal portfolio website. It is built to present my projects, technical skills, and background in Information Technology.

## Overview

The portfolio is a simple, responsive website developed using core web technologies. It serves as a central place to showcase my work and provide information about my growing interest in cybersecurity.

## Features

- Responsive layout for different screen sizes  
- Light and dark mode toggle  
- Project and skills section  
- Downloadable CV  
- Clean and minimal user interface  

## Technologies Used

- HTML  
- CSS  
- JavaScript  

## Project Structure

- `index.html` – Main structure of the website  
- `style.css` – Styling and layout  
- `script.js` – Interactivity and theme toggle  
- `Curriculum Vitae.pdf` – Downloadable resume  
- Images – Used within the site
